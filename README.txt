EVision

Library support for TFT display driver chips like the ST7735 and others, as
well as functions for drawing shapes, rendering text, and displaying bitmaps
on these screens.

Do note that this library is a work in progress, that the operation of this
software is not guaranteed correct, and the library functions may change in 
behavior and interface. A relatively stable version of this driver is planned
to be complete by August of 2024.
